# Class Books

## Member variables

The book
   * Author
   * Price
   * publisher
   * Title
   * Copies available
   * Cost per book
   * vector of books

## Member functions

`search()` - takes in title and author and returns the number of copies
available

`buy()` - buys x number of copies of the book

`add()` - adds inventory for a certain title

`display()` - displays the details of a book
